# مهارت‌های کتابخانه نقطه

این فایل توسط backend ساخته شده و لیست **همه** مهارت‌های قابل نصب در پنل نقطه را نشان می‌دهد.

- ✓ یعنی روی این پروژه نصب و فعال است (محتوای SKILL.md آن در همین پوشه قابل خواندن است).
- ◌ یعنی موجود در کتابخانه ولی روی این پروژه نصب نیست. کاربر می‌تواند از تب «مهارت‌ها → کتابخانه» در سایدبار پنل نصب کند.

**نکته برای ایجنت:** اگر کاربر پرسید «چه مهارت‌هایی داری؟» یا «چه کار می‌توانی بکنی؟»، **همه** آیتم‌های این فایل را نشان بده — نه فقط نصب‌شده‌ها — و بگو که برای فعال‌سازی آیتم‌های ◌ از UI استفاده کند.

## زیرساخت AI

- **✓ ساخت ابزار سفارشی (MCP)** (`mcp-builder`) — ساخت ابزار اختصاصی که نقطه بتواند مستقیم به سرویس‌های دلخواه شما (مثل بات تلگرام یا API خاص) متصل شود.

## دیپلوی و DevOps

- **✓ آروان‌کلود** (`arvancloud-platform`) — اتصال دامنه، CDN، SSL خودکار و میزبانی روی آروان‌کلود؛ سریع‌تر برای کاربر ایرانی و بدون تحریم.
- **✓ میزبانی روی Cloudflare** (`cloudflare-workers`) — اجرای سایت یا API روی Cloudflare Workers؛ رایگان و سریع برای مخاطب جهانی.
- **✓ ابزار Cloudflare CLI** (`cloudflare-wrangler`) — ابزار رسمی Cloudflare برای deploy و مدیریت Workers، دیتابیس و فایل.
- **✓ تغییر امن دیتابیس** (`db-migration-planner`) — تغییر ساختار دیتابیس روی سایت زنده بدون قطعی. مناسب پروژه‌هایی که یوزر فعال دارند.

## طراحی

- **✓ بهینه‌سازی سرعت سایت** (`cloudflare-web-perf`) — تحلیل سرعت بارگذاری سایت و پیدا کردن مشکلات کندی (تصاویر سنگین، CSS بلوک‌کننده و...).
- **✓ طراحی UI/UX حرفه‌ای** (`ui-ux-pro-max`) — کتابخانه طراحی با ۵۰+ سبک، صدها پالت رنگ و فونت‌بندی برای ظاهر شیک و مدرن. (نیاز: npm i motion)

## اسناد

- **✓ ساخت Word** (`docx`) — سند Word با عنوان، جدول و فرمت رسمی. مناسب نامه، قرارداد یا گزارش.
- **✓ ساخت PDF** (`pdf`) — نقطه می‌تواند فاکتور، گزارش یا هر فایل PDF برایت بسازد، PDF موجود را بخواند یا ویرایش کند.
- **✓ ساخت پاورپوینت** (`pptx`) — اسلایدشو با طراحی حرفه‌ای برای ارائه یا pitch محصول.
- **✓ ساخت Excel** (`xlsx`) — ساخت فایل اکسل با فرمول، نمودار و چند شیت. مناسب گزارش فروش یا لیست کالا.

## ایران

- **✓ فاکتور رسمی مالیاتی** (`invoice-sazgar`) — ساخت فاکتور رسمی سازگار با سامانه مودیان (QR کد، تاریخ شمسی، کد مالیاتی).
- **✓ درگاه پرداخت ایرانی** (`iran-payment-stack`) — اضافه‌کردن درگاه پرداخت (زرین‌پال، آیدی‌پی، زیبال و...) به پروژه بدون نیاز به دانش فنی.
- **✓ قوانین کسب‌وکار ایران** (`iranian-compliance`) — راهنمای گرفتن اینماد، نماد ساماندهی، نوشتن حریم خصوصی و قانون بازگشت کالا.
- **✓ فارسی‌سازی حرفه‌ای** (`persian-rtl-production`) — تقویم شمسی، اعداد فارسی، فونت زیبا و راست‌چینی درست برای سایت‌های ایرانی.
- **✓ سئو فارسی برای گوگل** (`persian-seo`) — بهینه‌سازی سایت برای رتبه‌گرفتن در گوگل ایران؛ متاتگ، sitemap و سرعت سایت.
