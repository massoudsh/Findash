# استراتژی کانورژن (Conversion)
### راهنمای کاربردی برای بازار آپشن ایران

---

## یک‌جمله‌ای بگویم چیست؟

> بازار یک اشتباه قیمت‌گذاری کرده. شما سه معامله همزمان می‌زنید و **سود قطعی** می‌گیرید — بدون اینکه اهمیت بدهد قیمت بالا می‌رود یا پایین.

---

## مفهوم به زبان ساده

فرض کنید در دو مغازه کنار هم، یک کالای یکسان قیمت‌های متفاوتی دارد. شما از مغازه ارزان می‌خرید و به مغازه گران می‌فروشید — سود تضمینی، بدون ریسک بازار.

کانورژن دقیقاً همین کار را در بازار آپشن انجام می‌دهد:

- **مغازه ارزان** = سهام + Put خریداری‌شده
- **مغازه گران** = Call فروخته‌شده
- **شرط سود** = قیمت‌گذاری بازار از تعادل خارج شده باشد

اگر بازار کاملاً کارآمد باشد، این فرصت وجود ندارد. اما وقتی قیمت‌گذاری Call و Put با هم «جور نباشند»، کانورژن سود می‌دهد.

---

## ساختار معامله

```
کانورژن = خرید سهام (Long Stock)
         + فروش اختیار خرید (Short Call)
         + خرید اختیار فروش (Long Put)

شرط:  قیمت اعمال و سررسید Call و Put باید یکسان باشند
```

---

## چرا سود تضمینی است؟

ترکیب Short Call + Long Put (با یک قیمت اعمال و سررسید) در واقع یک **فروش فوروارد ترکیبی (Synthetic Short Forward)** می‌سازد.

یعنی شما بدون توجه به اینکه قیمت سهام در سررسید چقدر باشد، **سهام را با قیمت اعمال می‌فروشید**:

| قیمت سهام در سررسید | چه اتفاقی می‌افتد |
|---------------------|------------------|
| بالاتر از قیمت اعمال | Call اعمال می‌شود — شما سهام را با قیمت اعمال می‌فروشید |
| پایین‌تر از قیمت اعمال | Put اعمال می‌شود — شما سهام را با قیمت اعمال می‌فروشید |
| برابر قیمت اعمال | هر دو بی‌ارزش منقضی می‌شوند — سهام را در بازار می‌فروشید |

**در هر حالت: درآمد شما = قیمت اعمال**

---

## معادله آربیتراژ

```
هزینه ایجاد موقعیت  = قیمت خرید سهام - پرمیوم Call + پرمیوم Put
درآمد سررسید        = قیمت اعمال (تضمینی)

سود آربیتراژ = قیمت اعمال - هزینه ایجاد موقعیت

شرط سود: قیمت اعمال > هزینه ایجاد موقعیت
```

این همان **Put-Call Parity** است:
```
C - P = S - PV(K)

C = پرمیوم Call
P = پرمیوم Put
S = قیمت جاری سهام
K = قیمت اعمال
PV(K) = ارزش فعلی قیمت اعمال (با نرخ بهره تنزیل می‌شود)
```

اگر این معادله برقرار نباشد → فرصت کانورژن وجود دارد.

---

## مثال عددی — بازار ایران

### مثال ۱: بدون فرصت آربیتراژ

صندوق طلای «زر» با قیمت **۱۰۰,۰۰۰ تومان** معامله می‌شود.

| معامله | مبلغ |
|--------|-------|
| خرید صندوق | ‎-۱۰۰,۰۰۰ تومان |
| فروش Call (اعمال ۱۰۰,۰۰۰) — پرمیوم | ‎+۶,۰۰۰ تومان |
| خرید Put (اعمال ۱۰۰,۰۰۰) — پرمیوم | ‎-۷,۰۰۰ تومان |
| **هزینه خالص** | **‎-۱۰۱,۰۰۰ تومان** |

درآمد تضمینی در سررسید: **۱۰۰,۰۰۰ تومان**

```
نتیجه: ۱۰۰,۰۰۰ - ۱۰۱,۰۰۰ = -۱,۰۰۰ تومان  ❌ زیان
```
بازار درست قیمت‌گذاری کرده — فرصتی نیست.

---

### مثال ۲: با فرصت آربیتراژ

همان صندوق، اما Put اشتباه قیمت‌گذاری شده:

| معامله | مبلغ |
|--------|-------|
| خرید صندوق | ‎-۱۰۰,۰۰۰ تومان |
| فروش Call (اعمال ۱۰۰,۰۰۰) — پرمیوم | ‎+۶,۰۰۰ تومان |
| خرید Put (اعمال ۱۰۰,۰۰۰) — پرمیوم | ‎-**۴,۰۰۰** تومان ← ارزان قیمت‌گذاری شده |
| **هزینه خالص** | **‎-۹۸,۰۰۰ تومان** |

درآمد تضمینی در سررسید: **۱۰۰,۰۰۰ تومان**

```
نتیجه: ۱۰۰,۰۰۰ - ۹۸,۰۰۰ = +۲,۰۰۰ تومان  ✅ سود بدون ریسک
```
Put ارزان‌تر از ارزش منصفانه قیمت‌گذاری شده — فرصت کانورژن!

---

## چطور فرصت را پیدا کنیم؟

```
اگر: Call - Put < S - K/(1+r)^t
→ فرصت کانورژن وجود دارد (Put ارزان است)

اگر: Call - Put > S - K/(1+r)^t
→ فرصت Reversal وجود دارد (برعکس کانورژن)
```

در بازار ایران، چون نرخ بهره بالاست، **Put باید گران‌تر از Call** باشد — اگر چنین نبود، احتمالاً فرصت وجود دارد.

---

## رایج‌ترین دلایل فرصت در بازار ایران

| دلیل | توضیح |
|------|--------|
| نقدشوندگی نامتعادل | حجم Call خیلی بیشتر از Put است — قیمت‌ها از هم فاصله می‌گیرند |
| نبود بازارگردان کارآمد | هیچ‌کس Put-Call Parity را فعالانه اجرا نمی‌کند |
| نرخ بهره بالا | اگر تنزیل به درستی در قیمت نباشد، انحراف پیدا می‌شود |
| فاصله bid-ask زیاد | قیمت واقعی اجرا از قیمت نظری فاصله دارد |

## محدودیت‌های عملی

### ۱. سرعت اجرا
سه معامله باید **تقریباً همزمان** اجرا شوند. اگر قیمت سهام در فاصله بین سه معامله تغییر کند، سود از بین می‌رود.

### ۲. کارمزد
در ایران، کارمزد معامله اختیار حدود ۰.۵٪ است. اگر سود آربیتراژ کمتر از کارمزدها باشد:

```
اگر سود < کارمزد خرید سهام + کارمزد Call + کارمزد Put → اجرا نکنید
```

### ۳. نقدشوندگی
برای بستن سه پوزیشن همزمان، باید در هر سه بازار **حجم کافی** وجود داشته باشد.

### ۴. نادر بودن فرصت
در بازارهای کارآمد، این فرصت‌ها کمتر از چند دقیقه دوام می‌آورند — معمولاً بانک‌ها و الگوریتم‌ها سریع‌تر هستند.

---

## مقایسه با Collar

| | کانورژن | کالر |
|--|---------|-------|
| **هدف** | آربیتراژ — سود بدون ریسک | محافظت از ریزش |
| **نگه داشتن سهام** | نه — در سررسید می‌فروشید | بله — می‌خواهید نگه دارید |
| **شرط اجرا** | قیمت‌گذاری اشتباه بازار | نگرانی از ریزش |
| **تکرار** | هر بار باید فرصت باشد | ماهانه قابل تمدید |
| **سختی اجرا** | بالا — نیاز به سرعت | متوسط |

---

## خلاصه سریع

```
✅ مناسب:  فرصت قیمت‌گذاری اشتباه پیدا کردید، سرمایه و سرعت اجرا دارید
❌ نامناسب: می‌خواهید سهام نگه دارید، کارمزدها بالاست، حجم بازار کم است

سود = قیمت اعمال - (قیمت سهام - پرمیوم Call + پرمیوم Put)
ریسک بازار = صفر (اگر درست اجرا شود)
ریسک اجرا = بالا (باید سه معامله همزمان باشند)
```

## چه کسی باید از این استراتژی استفاده کند؟

| سطح | مناسب؟ |
|-----|---------|
| مبتدی | ❌ خیر — پیچیدگی اجرا و ریسک کارمزد |
| متوسط | ⚠️ فقط برای یادگیری — فرصت واقعی نادر است |
| حرفه‌ای | ✅ با ابزار مناسب و دسترسی به داده آنی |
| الگوریتمی | ✅ بهترین گزینه — سرعت اجرا تضمین می‌شود |

> **نکته مهم:** کانورژن بیشتر از اینکه یک استراتژی معاملاتی عملی باشد، یک **ابزار درک بازار** است. فهمیدن آن به شما کمک می‌کند بفهمید چرا قیمت Call و Put همیشه با هم «حرکت می‌کنند».

---

## استراتژی‌های مرتبط

- **Reversal** — برعکس کانورژن: فروش سهام + خرید Call + فروش Put
- **Box Spread** — دو کانورژن روی دو قیمت اعمال مختلف = سود بدون ریسک کامل‌تر
- **Collar** — کانورژن بدون هدف آربیتراژ، با هدف محافظت از سهام
